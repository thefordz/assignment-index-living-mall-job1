# Pokemon list by generation

# Features

- Fetch Pokemon by generation
  - https://pokeapi.co/api/v2/generation/{generation_number}
- Fetch Pokemon details by name
  - https://pokeapi.co/api/v2/pokemon/{name}
- Grid layout (up to 16 Pokemon per row)
- Detail modal (name, image, abilities)
- Loading state (list & detail)
- Error handling with message
- Responsive design

# Setup

    1. npm install
    2. npm run dev

# TechStack

- Next.js 16
- Tailwind CSS
- Shadcn UI
